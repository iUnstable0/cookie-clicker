or run locally:

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Build:

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```

Run the production server:

```bash
npm start
# or
yarn start
# or
pnpm start
# or
bun start
```

Should be on port 3000
